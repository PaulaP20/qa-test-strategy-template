# QA Test Strategy Template

This repository contains a QA test strategy template designed for agile environments with multiple squads.

The goal is to provide a structured approach to quality assurance, covering testing strategy, risk assessment and release validation processes.

## Contents

- Test Strategy
- Risk Assessment
- Release Validation Checklist

## Context

This template reflects QA leadership practices used in multi-squad agile environments, where multiple teams collaborate to deliver features through coordinated release cycles.

It focuses on aligning QA, development, DevOps, and product teams to maintain consistent quality across distributed teams.
